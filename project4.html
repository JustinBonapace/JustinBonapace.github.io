Database Description Generator

This side project involved building an Excel‑based tool to automate and standardize component descriptions for various types of components such as motors, sensors, and fuses. Users select properties via dropdown menus, and the tool automatically generates both short schematic descriptions and detailed BOM descriptions. The project improved documentation speed, accuracy, and consistency, and is designed to be easily expandable for future component types.
